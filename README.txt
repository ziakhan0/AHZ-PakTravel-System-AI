## 📌 Project Overview

PakTravel AI System is an Artificial Intelligence-based project that integrates multiple AI techniques to solve real-world transportation problems. It includes route planning, knowledge-based reasoning, scheduling, prediction, and clustering using both classical AI algorithms and machine learning models.
